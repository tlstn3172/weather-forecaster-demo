# PRD: Weather Forecaster 앱

## 1. 프로젝트 개요

### 1.1 목적
사용자가 현재 위치 또는 특정 도시의 날씨 정보를 직관적이고 아름다운 인터페이스로 확인할 수 있는 모바일 우선 웹 애플리케이션을 개발합니다.

### 1.2 목표
- 실시간 날씨 정보 제공
- 시간별 및 10일간의 날씨 예보 제공
- 상세한 기상 정보 (UV 지수, 습도, 바람, 가시거리) 제공
- 프리미엄하고 현대적인 사용자 경험 제공
- 모바일 최적화된 반응형 디자인

### 1.3 타겟 사용자
- 일상적으로 날씨 정보를 확인하는 모든 사용자
- 여행 계획을 세우는 사용자
- 야외 활동을 계획하는 사용자
- 상세한 기상 정보가 필요한 사용자

## 2. 주요 기능

### 2.1 현재 날씨 정보
- **위치 기반 날씨**: 사용자의 현재 위치 자동 감지
- **현재 온도**: 대형 그라데이션 텍스트로 표시 (88px)
- **날씨 상태**: 아이콘과 텍스트로 표시 (맑음, 흐림, 비, 눈 등)
- **최고/최저 기온**: 일일 온도 범위 표시
- **날씨 아이콘**: Material Symbols를 활용한 동적 아이콘
  - 맑음: wb_sunny (노란색, 그림자 효과)
  - 부분 흐림: partly_cloudy_day
  - 흐림: cloud
  - 비: rainy (파란색)
  - 뇌우: thundererstorm (파란색)

### 2.2 시간별 예보
- **표시 범위**: 현재 시각부터 향후 6시간 이상
- **정보 항목**:
  - 시간 (Now, 1 PM, 2 PM 등)
  - 날씨 아이콘
  - 예상 온도
- **UI 특징**:
  - 수평 스크롤 가능한 카드 형식
  - 현재 시각 강조 표시 ("Now")
  - 반투명 배경 (backdrop-blur)

### 2.3 10일 예보
- **표시 범위**: 오늘 포함 10일간의 날씨
- **정보 항목**:
  - 요일 (Today, Tue, Wed 등)
  - 날씨 아이콘
  - 강수 확률 (해당 시)
  - 최저/최고 온도
  - 온도 범위 시각화 바 (그라데이션)
- **UI 특징**:
  - 그리드 레이아웃 (요일, 아이콘, 온도 범위)
  - 온도 범위를 색상 그라데이션 바로 표시
  - 강수 확률이 있는 경우 퍼센트 표시

### 2.4 상세 기상 정보

#### UV 지수
- 현재 UV 지수 수치
- 강도 레벨 (Low, Moderate, High 등)
- 시각적 그라데이션 바 (녹색→노란색→빨간색)
- 현재 위치 표시 마커
- 권장 사항 텍스트

#### 습도
- 현재 습도 퍼센트
- 이슬점 온도
- 체감 설명 텍스트

#### 바람
- 풍속 (m/s)
- 풍향 (방위)
- 나침반 시각화
- 방향 화살표 표시

#### 가시거리
- 가시거리 (km)
- 상태 설명
- 예상 정보

### 2.5 강수량 지도
- **지도 미리보기**: 현재 위치 중심의 날씨 지도
- **확대 기능**: "Open Map" 버튼으로 전체 화면 지도 보기
- **시각화**: 어두운 테마의 지도 스타일

### 2.6 도시 관리
- **도시 검색**: 특정 도시 이름으로 검색
- **도시 추가**: 관심 도시 저장
- **도시 목록**: 저장된 도시들의 날씨 한눈에 보기
- **도시 전환**: 탭으로 간편하게 도시 간 전환

### 2.7 위치 서비스
- **자동 위치 감지**: GPS 기반 현재 위치 자동 감지
- **위치 표시**: 헤더에 현재 도시명과 "Current Location" 표시
- **위치 아이콘**: location_on 아이콘과 파란색 인디케이터

## 3. UI/UX 요구사항

### 3.1 디자인 시스템

#### 색상 팔레트
- **Primary**: #2b8cee (파란색)
- **Background (Dark)**: #101922 (깊은 다크 블루-그레이)
- **Card Background**: #1c252e (카드용 약간 밝은 톤)
- **Background (Light)**: #f6f7f8 (라이트 모드용)
- **Accent Colors**:
  - 노란색: 맑은 날씨 (#fbbf24)
  - 파란색: 비/뇌우 (#60a5fa)
  - 그라데이션: 온도 표시용 다양한 색상

#### 타이포그래피
- **폰트 패밀리**: Inter (Google Fonts)
- **폰트 크기**:
  - 현재 온도: 88px (thin weight)
  - 섹션 제목: 12-14px (semibold, uppercase)
  - 본문: 14-16px
  - 캡션: 10-12px

#### 아이콘
- **아이콘 시스템**: Material Symbols Outlined
- **아이콘 크기**: 
  - 메인 날씨: 84px
  - 시간별/일별: 24px
  - 네비게이션: 24-28px
- **아이콘 스타일**: Fill variation 활용

#### 레이아웃
- **최대 너비**: 448px (모바일 중심)
- **패딩**: 20px (기본)
- **카드 간격**: 16-24px
- **Border Radius**: 
  - 카드: 24px (3xl)
  - 버튼: 9999px (full)
  - 작은 요소: 12px (xl)

### 3.2 시각적 효과

#### 배경
- **앰비언트 그라데이션**: 상단에서 하단으로 primary 색상 그라데이션 (20% → 5% → 투명)
- **다크 모드**: 기본 다크 테마
- **블러 효과**: 카드에 backdrop-blur 적용

#### 그라데이션
- **온도 텍스트**: 흰색에서 연한 파란색으로 (180deg)
- **온도 범위 바**: 
  - 차가운 날씨: 파란색 → 녹색
  - 따뜻한 날씨: 녹색 → 노란색
  - 더운 날씨: 노란색 → 주황색

#### 인터랙션
- **호버 효과**: 버튼에 배경색 변화
- **트랜지션**: 부드러운 색상 전환
- **스크롤**: 커스텀 스크롤바 숨김 (모바일 룩)

### 3.3 네비게이션

#### 상단 헤더
- **왼쪽**: 메뉴 버튼 (햄버거)
- **중앙**: 현재 위치 표시
  - 위치 아이콘
  - 도시명
  - "Current Location" 인디케이터
- **오른쪽**: 설정 버튼

#### 하단 탭 바
- **Weather**: 메인 날씨 화면 (활성화 시 primary 색상)
- **Cities**: 도시 목록
- **Radar**: 레이더 지도
- **Profile**: 사용자 프로필/설정

### 3.4 반응형 디자인
- **모바일 우선**: 최대 448px 너비로 제한
- **스크롤 최적화**: 부드러운 수직 스크롤
- **터치 친화적**: 충분한 터치 영역 (최소 44px)

## 4. 기술 요구사항

### 4.1 프론트엔드
- **프레임워크**: React / Next.js 또는 Vanilla JavaScript
- **스타일링**: Tailwind CSS
- **아이콘**: Material Symbols
- **폰트**: Google Fonts (Inter)

### 4.2 API 통합
- **날씨 API**: OpenWeatherMap, WeatherAPI, 또는 유사 서비스
- **위치 API**: Geolocation API (브라우저 내장)
- **지도 API**: Google Maps 또는 Mapbox

### 4.3 데이터 요구사항
- **현재 날씨**: 온도, 날씨 상태, 최고/최저 기온
- **시간별 예보**: 최소 24시간 데이터
- **일별 예보**: 10일 데이터
- **상세 정보**: UV 지수, 습도, 풍속, 풍향, 가시거리, 이슬점
- **강수 정보**: 강수 확률, 강수량

### 4.4 성능 요구사항
- **초기 로딩**: 3초 이내
- **API 응답**: 2초 이내
- **부드러운 애니메이션**: 60fps
- **오프라인 지원**: 마지막 데이터 캐싱

### 4.5 브라우저 지원
- **모바일**: iOS Safari 14+, Chrome Mobile 90+
- **데스크톱**: Chrome 90+, Firefox 88+, Safari 14+, Edge 90+
