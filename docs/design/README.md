# Design Files

이 폴더에는 Weather Forecaster 앱의 디자인 파일들이 포함되어 있습니다.

## 파일 목록

### `screen.png`
- 앱의 메인 화면 디자인 스크린샷
- 다크 테마 기반의 날씨 앱 UI
- 현재 날씨, 시간별 예보, 10일 예보, 상세 정보 포함

### `code.html`
- 디자인 구현 HTML 코드
- Tailwind CSS 사용
- Material Symbols 아이콘 사용
- 모바일 최적화 레이아웃 (최대 448px)
